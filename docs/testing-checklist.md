# 手工测试清单

## 订阅管理
- [ ] 新建 SEARCH 订阅 → 保存后自动弹出检查结果
- [ ] 新建 SPECIFIC_GALLERY 订阅（粘贴 exhentai 画廊 URL）→ 检查并下载
- [ ] 编辑订阅：修改名称/间隔/分卷/下载目录/筛选条件 → 保存后生效
- [ ] 订阅列表搜索：输入名称 → 筛选结果；按源/状态过滤
- [ ] 删除订阅（确认无报错）

## 订阅检查与去重
- [ ] SEARCH 检查 → 显示新画廊卡片（有封面/标题/页数/标签）
- [ ] "加载更多"翻页 → 去重生效
- [ ] 已下载画廊不在新结果中出现
- [ ] SPECIFIC_GALLERY 增量更新：添加新页后再次检查 → 只显示新页
- [ ] 筛选条件生效：设置"页数≥20" → 小画廊不出现

## 下载
- [ ] 批量选中下载 → 队列页显示等待→下载中→完成
- [ ] 分卷下载（cbz_max_pages>0）→ CBZ 正确拆分
- [ ] 不分卷下载（cbz_max_pages=0）→ 单 CBZ 含所有页
- [ ] 增量下载→ 新页追加到已有 CBZ（旧 CBZ 被重打包）
- [ ] 下载失败→ 队列显示失败状态 + 标题/封面不丢 + 可重试
- [ ] 重试后失败条目从失败区消失

## CBZ 内容
- [ ] 查看 CBZ 内页数是否正确
- [ ] ComicInfo.xml 含 Title/Writer/Year/Month/Day/Web/Tags
- [ ] 标签翻译为中文
- [ ] CBZ 文件名格式：`[{id}] {标题} ({起}-{止}).cbz`

## 通知
- [ ] 下载完成→ 收到邮件（含封面/标题/页数/链接）
- [ ] 部分失败→ 邮件含成功/失败汇总
- [ ] 下载完成后 Komga 触发扫描

## 画廊管理
- [ ] 画廊列表显示已下载画廊
- [ ] 按源/排序筛选正常
